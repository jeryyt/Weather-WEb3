How to Use

  1. Get an API Key: Sign up at OpenWeatherMap to get your free API key
  2. Configure Widget: Fill in the form above with your preferences
  3. Generate Code: Click "Generate Embed Code" to create your custom embed code
  4. Copy & Paste: Copy the generated code and paste it into your website's HTML
  5. Host Widget: Make sure to host the widget.html file on your server
